# Jira / Program Update Summarizer

This project converts raw Jira ticket content into concise,
executive-ready summaries using a GenAI model.

## Use Cases
- Weekly program updates
- Incident summaries
- Leadership reviews
- Status reporting

## Highlights
- Uses pre-trained transformer model
- Secure model loading with SafeTensors
- Designed for TPM workflows

## How to Run
```bash
python jira_summarizer.py
